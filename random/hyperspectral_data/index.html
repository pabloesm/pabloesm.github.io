Dimensionality reduction

Real-life problems naturally involve high-dimensional feature vectors. In most cases, the features contain redundant information, which means that they are not independent from each other. The large set of features can be summarized into a smaller set, with no or less redundancy. This is the goal of dimensionality reduction (DR) methods, which play an important role in the fields of data analysis, data mining, and machine learning.

The following example shows a scatter plot that represent a set of 189 mandarins, some of which present decay symptoms caused by Penicillium digitatum fungus. For each mandarin, 1569 features were measured using a spectroscopy system. These features were processed using Principal Component Analysis (PCA), a popular method of dimensionality reduction.

The graph enables to compare the PCA features with any pair of the original features. It can be observed that PCA features provide a representation that facilitates tasks such as the automatic detection of decayed fruit.
